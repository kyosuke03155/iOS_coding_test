# iOS 未経験者エンジニア向けコードチェック課題

## アプリ仕様
- 「あなたと相性のいい都道府県を占ってあげる！」をテーマに iPhone アプリ  
- 名前、生年月日、血液型を入力とし、都道府県を出力する
